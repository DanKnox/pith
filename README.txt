Pith
====

Pith is a static web-site generator.

Features
--------

* Haml and Sass templates, via [Tilt][tilt].
* Layouts, and nested layouts.
* Inclusion of "partials".
* Easy linking between pages.
* Ability to define custom helper-methods.
* Built-in web-server for testing.
* Incremental rebuilding (doesn't re-generate outputs unless the corresponding inputs have changed).

[tilt]: http://github.com/rtomayko/tilt/
